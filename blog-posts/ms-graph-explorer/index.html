Microsoft Graph Explorer

Microsoft Graph provides a unified endpoint to interact with data in Microsoft services such as Azure Active Directory, Office 365 services, Enterprise Mobility and Security services, Windows 10 services, Dynamics 365, and more.

We created the Microsoft Graph Explorer for developers working with the API. The new webpage contains powerful features to allow both consumer and commercial services to explore the API. Contributions include an autocomplete tool, access for work and school accounts, query history, and customized request headers.
